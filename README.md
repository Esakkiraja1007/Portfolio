# 🚀 Modern Frontend Developer Portfolio

Welcome to my personal portfolio website repository! This project serves as my digital home, showcasing my tech stack, recent web development projects, and services as a Frontend Developer. 

✨ **Live Demo:** [Esakki Raja Portfolio](https://esakkiraja1007.github.io/Portfolio/)

---

## 🛠️ Features

* **Modern Design System:** Built with a beautiful layout using **Outfit** (for headings) and **Inter** (for body text).
* **Pure CSS Light/Dark Mode:** Dynamic theme toggling achieved using advanced CSS logic (`body:has(#theme-toggle:checked)`) without needing external heavy libraries.
* **100% Responsive Design:** Smooth, flexible layout transitions optimized perfectly for desktop, tablet, and mobile devices.
* **CSS-Only Mobile Menu:** Fully functional mobile navigation handled beautifully via the checkbox hack method.
* **Glassmorphism UI:** Clean glass-panel styling with subtle backdrop filters, gradients, and hovering effects.
* **SEO & Social Optimization:** Pre-configured Meta tags, Open Graph (Facebook), and Twitter Card configurations along with an automated `sitemap.xml` and `robots.txt` structure.
* **Functional Contact Form:** Fully integrated using Formspree for direct email communication.
